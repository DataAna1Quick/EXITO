# Operación Éxito Antioquia — Reporte Q1 2026

Reporte interactivo (HTML estático) de la operación logística Quick para Éxito Antioquia: posicionamiento de flota, fidelización (Loyalty), riesgo de recaudo en efectivo por macrozona, escoltas operativos y hoja de ruta Q2 2026.

## Ver el reporte

Publicado con GitHub Pages: https://dataana1quick.github.io/EXITO/

## Navegación

- Flechas del teclado / swipe para pasar slides
- Click en los puntos inferiores para saltar a una slide específica
- 29 slides distribuidas en módulos: Posicionamiento, Loyalty, Riesgo de Efectivo (9 fichas de zona), Escoltas, Estrategia
